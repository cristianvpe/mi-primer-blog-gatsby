# Que es API rest
ES UNA INTERFAZ PARA COMUNICAR ENTRE APLICACIONES
- [apirest.com](apirest.com)

Una API REST (Interfaz de Programación de Aplicaciones REST por sus siglas en inglés) es un tipo de interfaz que permite que las aplicaciones se comuniquen entre sí de forma estandarizada.  Imagina que es un mesero en un restaurante, la API REST sería el menú que le permite a los clientes (otras aplicaciones) saber qué platillos hay disponibles (recursos) y cómo pedirlos (operaciones).

Aquí te explico algunos aspectos clave de las API REST:

Arquitectura Cliente-Servidor: Las API REST siguen un modelo cliente-servidor, donde una aplicación cliente (el que pide los datos) se comunica con una aplicación servidor (la que tiene los datos) para obtener información o realizar acciones.
Verbos HTTP: Las API REST utilizan verbos HTTP estándar como GET (obtener), POST (crear), PUT (actualizar) y DELETE (eliminar) para realizar operaciones sobre los datos. Por ejemplo, si un cliente quiere obtener una lista de usuarios, usaría una petición GET.
Recursos sin estado: Las API REST están diseñadas para ser sin estado, lo que significa que cada petición del cliente debe incluir toda la información necesaria para procesarla. El servidor no guarda información sobre las peticiones anteriores del cliente.
Formatos de datos: Las API REST suelen utilizar formatos de datos como JSON o XML para intercambiar información entre el cliente y el servidor.
Las API REST son muy populares porque son flexibles, ligeras y fáciles de usar. Se utilizan en una amplia variedad de aplicaciones, desde aplicaciones móviles hasta servicios web.
