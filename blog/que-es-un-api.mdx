# Que es GraphQL
Es una interfaz de aplicaciones web
Una API (Interfaz de Programación de Aplicaciones) es como un mensajero que permite que diferentes programas informáticos se comuniquen entre sí e intercambien información.  Imagina que tienes dos aplicaciones:

Una que muestra el tiempo (la aplicación del tiempo).
Otra que te permite planear tu viaje (una app de viajes).
La aplicación del tiempo no necesita tener toda la información meteorológica del mundo almacenada. En su lugar, puede utilizar una API para solicitar datos meteorológicos a un servicio meteorológico online. La API actúa como intermediario, llevando la petición de la aplicación del tiempo al servicio meteorológico y devolviendo la información solicitada.

Aquí te resumo algunos puntos clave sobre las API:

Integración de sistemas: Las API facilitan la conexión entre distintos programas, permitiendo que compartan funcionalidades y datos.
Reutilización de código: Los desarrolladores pueden crear funciones y acceder a información a través de las API, evitando programar todo desde cero.
Flexibilidad: Las API permiten construir aplicaciones más complejas y dinámicas.
Las API son esenciales en el mundo actual, donde las aplicaciones web y móviles necesitan comunicarse e intercambiar datos constantemente.
